4 Axis Robot

3D Design, Motor Control, C++ and Python

What?
A robotic arm designed completely from scratch. Programmed the robot to place blocks in squares 1-9 of a tic-tac-toe board as a test of its functionality. I also programmed tic-tac-toe logic so that the robot could play against a user.
How?
Used Fusion 360 to model all the components, which were then 3D printed. An Arduino board was used to control the micro servo motors. Programmed motor control with C++ and the game logic with python.
Results?
The robot could process the user's chosen position, decide its next move, pick up a block, and accurately place it. The robot's logic ensured it was unbeatable in gameplay.

Automated Culinary Robotics System

Mechatronics Design

What?
A robot designed to create a customized cup of pudding for a quarter by combining pudding powder, water, and sprinkles. Depending on user specifications, the robot could make different types of pudding.
How?
Programmed user input, sensor logic, and motor control using C++. Built the robot using LEGO Technic components and 3D-printed parts modeled in Fusion 360.
Results?
Built a fully functional robot capable of detecting a quarter, processing user input for flavor preferences, accurately moving cups via conveyor, dispensing ingredients, and mixing the pudding. Earned recognition from the professor as the most intricate project in the class.

Filtration System
for Remote Communities

Project Management

What?
Led a team of 5 to design a water filtration system for underserved northern communities, aiming to produce over 100L/day of clean water.
How?
Programmed an Arduino-controlled chlorination and agitation system in C++ for one-touch disinfection; modeled and 3D printed compact electronic housing in Fusion 360 for seamless pump integration.
Results?
Delivered clear, drinkable water from heavily soiled sources, surpassing the 100L/day goal. Project was recognized as the top performer in the cohort for filtration efficiency and system reliability.
